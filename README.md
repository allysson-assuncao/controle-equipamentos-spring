Instruções de Instalação e execução do projeto:

Instalação: Basta clonar o repositório e deixar o SpringBoot baixar as dependências do backend.

Execução: Rodar o arquivo Application do SpringBoot no backend e rodar o frontend com o comando "python3 -m http.server 3000". O banco local H2 Database é configurado e gerenciado automaticamente.
Para acessar a aplicação web basta abrir a página "http://127.0.0.1:3000/index.html".
